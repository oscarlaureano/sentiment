# sentiment

### Instalación

Tecnología:
- python3

Librerias requeridas:
- Flask

`pip install flask`

- Dash

`pip install dash==1.4.0 `

`pip install dash-daq==0.2.1 `

- Tweepy

`pip install tweepy `

- Pandas

`pip install pandas `

- TextBlob

`pip install textblob`

### Para correr

` python dashApp.py `
